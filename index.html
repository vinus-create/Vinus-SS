            How It Works
            • 1️⃣
              Browser crawls Shopee category pages
              Uses your session cookies — no CAPTCHA for normal speeds
            • 2️⃣
              Extracts every shop selling in that category
              Product data + shopid saved automatically
            • 3️⃣
              New shops appear in the Market tab
              Next daily cron scrapes their full product list
            • 4️⃣
              Compare them in Best Sellers, Compare tabs
              Full variant + velocity data after first cron run

            Full Setup Guide

            Supabase + Vercel + Claude in Chrome — complete production stack

            1
            Supabase
            Create project, run SQL schema, copy URL + service key
            2
            GitHub
            Push shopee-scope folder, connect repo to Vercel
            3
            Vercel
            Add env vars, deploy — get your live dashboard URL
            4
            Scrape
            Open shopee.com.my in Chrome, run scraper.js via Claude
            5
            Analyse
            Open dashboard on phone, select shop, get insights
            1. Supabase Setup
            • 1️⃣
              Go to supabase.com → New Project
            • 2️⃣
              SQL Editor → paste supabase_schema.sql
            • 🔑
              Settings → API → copy Project URL and service_role key
              NOT the anon key — service key bypasses RLS
            • ✅
              Tables: shops, products, scrape_log
            2. Vercel Deploy
            • 1️⃣
              Push project to GitHub repo
            • 2️⃣
              vercel.com → Import Git Repository
            • 🔑
              Environment Variables:
              SUPABASE_URL=https://xxx.supabase.co
              SUPABASE_SERVICE_KEY=your_service_key
            • 🌐
              Your live URL: your-app.vercel.app
            3. Run Scraper
            • 1️⃣
              Open Chrome with Claude in Chrome extension
            • 2️⃣
              Navigate to shopee.com.my
            • 3️⃣
              Edit scraper.js — set VERCEL_URL + shop usernames
            • 4️⃣
              Ask Claude to run the scraper via javascript_tool
            • 📱
              Open dashboard on phone → select shop → done!
