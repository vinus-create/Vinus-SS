            ⚙️ Setup & Architecture

            Full deployment guide for Supabase + Vercel + Claude in Chrome

            1
            Supabase
            Create project, run SQL schema, copy URL + service key
            2
            GitHub
            Push shopee-scope folder, connect repo to Vercel
            3
            Vercel
            Add env vars SUPABASE_URL + SERVICE_KEY, deploy
            4
            Scrape
            Open shopee.com.my in Chrome, ask Claude to run scraper
            5
            Analyse
            Open dashboard on phone, select shop, get insights
